Création de liste d'options à partir d'un tableau

    Créez une page td3form.php contenant :
  • Un formulaire (méthode GET, action td3affiche.php)
  • Une liste déroulante pour choisir un jour (1 à 31)
  • Une liste déroulante pour choisir une année (1980 à 2005)
  • Une liste déroulante pour choisir un mois (valeur = nom du mois)
Modification : Faites transmettre le numéro du mois (ex: 2 pour Février) *Page td3affiche.php doit afficher :* "La date choisie est le 3/5/1993"
